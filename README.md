# Ed's Nix Config

<img align="center" src="https://raw.githubusercontent.com/catppuccin/catppuccin/main/assets/palette/macchiato.png">

## 🚀 Features

- Based on [snowfall library](https://github.com/snowfallorg/lib).
- NixOS configurations featuring [Hyprland window manager](https://hyprland.org/) with [waybar](https://github.com/Alexays/Waybar) menu and
  [anyrun](https://github.com/anyrun-org/anyrun) launcher.
- [NixColors](https://github.com/Misterio77/nix-colors)-based themeing for quick switching of colour palette through all apps.
- [Catppuccin](https://github.com/catppuccin/catppuccin) theme.
- NeoVim editor configuration through [NixVim](https://github.com/nix-community/nixvim).
